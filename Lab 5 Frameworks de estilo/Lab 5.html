¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Hay varias diferencias pero las principales son:

  • Javascript es un lenguaje interpretado, donde Java se compila
  • Javascript es un lenguaje dinámico, mientras que Java tiene variables definidas
  • Javascript es un lenguaje basada en prototipos, mientras que Java está basada en objetos.
  • Javascript es débilmente tipado, mientras que Java es fuertemente tipado

Las similitudes que tiene aparte del nombre, es que ambos son lenguajes de programacion de alto nivel y estan basados en C. Fuera de eso, no tienen muchas similitudes, aparte de que usan conceptos de POO como Los dos utilizan conceptos como encapsulación, herencia, polimorfismo y abstracción.

Cerón, M. (23 de Mayo, 2023). ¿Qué diferencias hay entre JavaScript y Java? https://www.hackaboss.com/blog/diferencias-javascript-java

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

El objeto de Date tiene muchos métodos pero unos ejemlos son:

  • Date.now() - Devuelve el valor númerico correspondiente al actual número de milisegundos transcurridos desde el 1 de Enero de 1970, 00:00:00 UTC, ignorando los segundos intercalares.
  • Date.prototype.getDate() - Retorna el día del mes (1 - 31) para la fecha especificada acorde a la hora local.
  • Date.prototype.getTimezoneOffset() Retona la diferencia horaria en minutos para la hora local.
  • Date.prototype.toString() - Retorna una cadena representando el objeto especificado Date.
  • Date.prototype.setHours() - Establece la hora para una fecha especifica acorde a la hora local.
  • Date.prototype.getMinutes() Retorna los minutos (0 - 59) para la fecha especificada acorde a la hora local.

MDN Web Docs. (s.f.) Date. https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Global_Objects/Date

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

Los arreglos también tienen muchos métodos distintos, y aquí hay varios ejemplos:

  • push() - Añade un elemento al arreglo
  • unshift() - Añade un elemento al principio del arreglo
  • pop() - Elimina el último elemento del arreglo
  • shift() - Elimina un elemento al principio del arreglo
  • slice() - Copia y clona un arreglo a otro arreglo
  • Array.isArray(value) - Regresa true or false y ve si es un arreglo

Zubikarai, S. (12 de enero, 2022). Manual de arreglos en JavaScript: Métodos de arreglos de JS explicados con ejemplos. https://www.freecodecamp.org/espanol/news/el-manual-de-arreglos-en-javascript/

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable local en una función, solo hay que usar una de las palabras claves como let, var y const. De esta forma, si se intenta acceder a la variable fuera de la función no se puede.

Uniwebsidad. (s.f.). 4.2. Ámbito de las variables https://uniwebsidad.com/libros/javascript/capitulo-4/ambito-de-las-variables

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

La recomendación es siempre usar variables locales para funciones, ya que si se usa una variable global, estas variables se pueden compartir entre funciones y hasta modificar, y esto puede llegar a no ser seguro para una página web.

Durán, M. (13 de Octubre, 2022). Qué es una variable global en JavaScript y cómo se declara. https://blog.hubspot.es/website/variable-global-javascript
